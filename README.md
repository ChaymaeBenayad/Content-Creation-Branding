# Branding landing page

Branding is a simple, beautiful and responsive landing page dedicated for agencies that aim to help companies to elevate their visual branding to attract more clients by offering the best services in photography, filmmaking, video editing and social media management. It is developed using HTML5, CSS3, JavaScript and jQuery library.
